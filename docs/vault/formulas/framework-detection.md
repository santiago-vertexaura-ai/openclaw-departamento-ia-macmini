METODO para detectar que framework de marketing usa una pieza de contenido viral:

CHECKLIST:
- Empieza con resultado especifico? -> Hormozi (dream outcome)
- Cuenta una historia personal? -> Brunson (story selling)
- Usa piramide (1 pieza -> muchas)? -> Gary Vee (content pyramid)
- Apela a un nivel de awareness especifico? -> Schwartz
- Usa urgencia/escasez? -> Dan Kennedy
- Tiene hook numerico? -> Hormozi/Vadim pattern

CUANDO USAR: En cada analisis de contenido viral. Tag el framework en el output JSON. Permite a Marina replicar la formula exacta.

Relacionado con [[schwartz-awareness-analysis]], [[hormozi-twitter-hooks]].
