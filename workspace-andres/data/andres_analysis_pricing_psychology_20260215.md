- **Mecanismo psicológico:** Método Socrático (simplificación) + Anchor effect (49€ y 50€+ son anchors para validar 47€).

**Hook 2: "Left Digit Effect — los cerebros procesan primero el primer dígito. 49€ se procesa neuralmente como '40-algo'"**

- **Por qué funciona:** EXPLICA EL MECANISMO con nombre científico. No es "49€ parece barato", es "49€ se procesa como '40-algo' porque el Left Digit Effect". Naming + mechanism revelation = credibilidad + curiosidad simultáneamente.
- **Mecanismo psicológico:** Appeal to authority + Demystification (revelar el truco hace que sea más convincente, no menos).

## CAPA 2: FÓRMULAS REPLICABLES — PLANTILLAS PARA FUTUROS PROYECTOS

Roberto extrae **5 fórmulas clave** reutilizables:

### 1. ODD_PRICING_EDUCATED_AUDIENCES
**Cuándo:** Audiencias educated (developers, creadores) que valorizan exclusividad sobre precio bajo.  
**Plantilla:** Identifica audiencia → Descarta X9 → Selecciona X7 (47, 97, 297) → Valida sin competencia directa → Usa mensajería "sofisticado, no obvio".  
**Ejemplo:** Suite Pro OpenClaw a 197€/ms (no 199€).

### 2. FOUNDER_PRICING_URGENCY_ENGINE
**Cuándo:** Lanzamientos nuevos necesitando volumen + revenue + customer quality.  
**Plantilla:** Precio base → Descuento 40-45% para primeros 100-200 → Límite explícito → Comunicación cambio de precio → Monitorea 80% adoption → Intensifica urgencia a 60% del cap.  
**Ejemplo:** OpenClaw Certification 297€, founder pricing 177€ (40% desc) primeros 50.

### 3. PRICING_DINAMICO_ESCALONADO
**Cuándo:** Crecimiento gradual + urgencia continua sin big bang.  
**Plantilla:** 3 tiers tiempo (FOUNDER, EARLY, REGULAR) → Precios escalonados (26→47→79€) → Comunicación precio congela → Tabla pública → FOMO + scarcity visible.  
**Ejemplo:** Semana 1-8: 26€, Semana 9-16: 47€, Semana 17+: 79€.

### 4. MONEY_BACK_GUARANTEE_CONVERSION_LEVER
**Cuándo:** Fricción de decisión (nueva categoría, audiencia dudosa).  
**Plantilla:** 30-días reembolso → Annual 22% discount → Mensajería riesgo cero → Target 40-50% monthly→annual → Churn <5%.  
**Ejemplo:** Certification MBG 30 días, annual (22% desc): 2,340€ vs 3,000€.

### 5. CHARM_VS_ODD_VS_PRESTIGE_SELECTION_FRAMEWORK
**Cuándo:** SIEMPRE. Framework universal de pricing.  
**Plantilla:** Categoriza audiencia (price-sensitive, value-conscious, premium) → Mapea competencia → Test X9, X7, X0 → Mide conversión, LTV, NPS → Ganador = máximo LTV.
