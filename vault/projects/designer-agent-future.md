PROYECTO APARCADO: Agente disenador para generar visuales automaticos.

IDEA: Un agente que genere imagenes, graficos y visuales para el contenido de Marina. Usaria modelos de generacion de imagen (FLUX, SDXL local, o APIs como Ideogram/Midjourney).

HERRAMIENTAS CONFIRMADAS:
- Higgsfield (motion control, character swap, generacion video con cara de Santi)
- Nano Banana Pro (generacion de imagenes, accesible desde Higgsfield)
- Gamma (presentaciones)
Todas tienen API.

CASOS DE USO:
- Marina pide a Designer que genere imagenes para posts
- Santi puede replicar reels virales con su cara
- Otros agentes dan instrucciones al Designer para contenido visual

BLOQUEANTES ACTUALES:
- Modelos de imagen local requieren GPU (Mac Mini no tiene GPU dedicada suficiente)
- APIs de imagen tienen coste
- Prioridad baja vs Alex y mejoras del pipeline actual

DECISION: Reavaluar cuando el pipeline actual (Roberto->Andres->Marina) funcione fluido y Skool este lanzado.
