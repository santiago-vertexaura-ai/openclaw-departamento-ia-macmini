# Hook Adaptation — Adaptar hooks a plataformas específicas

## Cuándo usar
- task_type: `hook_adaptation`

## Input
- Hook(s) existentes a adaptar
- Plataforma(s) destino
- Contexto/audiencia

## Output
Para cada hook x plataforma:
1. Hook adaptado respetando constraints de la plataforma
2. Justificación de por qué funciona en esa plataforma
3. A/B test sugerido (variante alternativa)

## Reglas
- LinkedIn hooks: profesionales, thought leadership, datos
- Twitter hooks: punchy, controversial takes, números
- Instagram hooks: emocionales, storytelling, visual-first
