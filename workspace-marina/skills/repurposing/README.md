# Repurposing — Reutilizar contenido existente en nuevos formatos

## Cuándo usar
- task_type: `repurposing`

## Input
- Contenido original (post, artículo, doc)
- Formatos destino (thread, carrusel, reel script, newsletter)
- Plataformas destino

## Output
Para cada formato/plataforma:
1. Contenido reformulado (NO copy-paste)
2. Hooks nuevos adaptados al formato
3. CTA adaptado
4. Visual brief si aplica

## Reglas
- Cada re-formato debe aportar algo nuevo (no solo recortar)
- Adaptar al lenguaje nativo de cada plataforma
- Mantener los datos clave del original
- Hooks diferentes al original (no repetir)
